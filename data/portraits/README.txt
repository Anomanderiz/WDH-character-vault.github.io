Manual portrait overrides live here.

Naming convention:
- <Character Name>-img.<ext>

Examples:
- Goody-img.png
- Meredith Vixeni-img.webp

Supported file extensions:
- .webp
- .png
- .jpg
- .jpeg
- .avif

Notes:
- If exact character-name matching fails, the viewer also tries a slug form:
  e.g. "Kira Kira Eventide-img.png" and "kira-kira-eventide-img.png".
